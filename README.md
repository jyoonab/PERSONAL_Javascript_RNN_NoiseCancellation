# RNNoise with WebRTC

## Purpose
This repository is to demonstrate how well the AI-based noise reduction works.
Currently, We are planning to demonstrate this by using :

1. Live Chat(WebRTC)
2. Sample Sound

## Requirement
Node > 14.15.4

Npm > 6.14.10

## Installation & Usage
```
$ npm install
$ npm start
```
